# ExpressBoilerplate (v1.0.0)

A simple boilerplate for express

## Project setup

```
npm install
```

### Run for development

Firstly, you need to create a development configuration file, at:

```
./src/config.dev.js
```

Then you can run in development mode by using:

```
npm run dev
```

### Run for production

```
npm run prod
```
